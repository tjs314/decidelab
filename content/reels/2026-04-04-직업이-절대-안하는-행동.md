# 릴스 기획안 | 2026.04.04

## 컨셉: [직업]이 절대 안 하는 [행동] N가지

> **목적**: 직장인 공감 콘텐츠로 유입 → DecideLab 커리어 진단 전환
> **타깃**: 직장 3~7년차, 이직/창업 고민 중인 직장인
> **톤앤매너**: 유머 + 찔림 (웃기지만 "나인데?" 반응 유도)

## 안 1) 퇴사 고민러가 절대 안 하는 5가지

**공감 포인트**: DecideLab 타깃과 직접 연결 (이탈 신호 B섹션)

| 순서 | 화면 텍스트 | 비고 |
|------|------------|------|
| Hook | "퇴사 고민하는 사람 특징" | 0~1초, 강렬한 텍스트 |
| 1 | 월요일 아침에 알람 한 번에 일어나기 | B섹션 Q1 연결 |
| 2 | 연봉 계산기 안 돌려보기 | B섹션 Q3 연결 |
| 3 | "5년 뒤에도 여기 있겠지~" 하고 안심하기 | B섹션 Q4 연결 |
| 4 | 링크드인 프로필 안 만지기 | B섹션 Q5 연결 |
| 5 | 친구 이직 소식에 무반응 | B섹션 Q7 연결 |
| CTA | "3개 이상 해당되면 진단 한번 받아봐" | DecideLab 링크 |

- **예상 길이**: 15~20초
- **BGM**: 트렌디한 밈 사운드 / 긴장감 있는 비트
- **자막 스타일**: 큰 텍스트 + 카운트다운

## 안 2) 회사 잘 다니는 사람이 절대 안 하는 4가지

**공감 포인트**: 직장 적합도 A섹션과 반전 연결

| 순서 | 화면 텍스트 | 비고 |
|------|------------|------|
| Hook | "회사 잘 다니는 사람은 이거 절대 안 함" | 반전 기대감 |
| 1 | 성장하고 있다는 착각 없이 버티기 | A섹션 Q1 역발상 |
| 2 | 연봉 시장가를 모른 채로 1년 보내기 | A섹션 Q4 연결 |
| 3 | 배울 사람 없는 팀에서 혼자 고군분투 | A섹션 Q6 연결 |
| 4 | "내년엔 나아지겠지" 하고 방치하기 | A섹션 Q8 연결 |
| CTA | "혹시 나? 싶으면 30문항 체크해봐" | DecideLab 링크 |

- **예상 길이**: 12~15초
- **포맷**: 텍스트 오버레이 + 표정 리액션

## 안 3) 창업 준비된 사람이 절대 안 하는 3가지

**공감 포인트**: 창업 준비도 C섹션 핵심 연결

| 순서 | 화면 텍스트 | 비고 |
|------|------------|------|
| Hook | "진짜 창업할 사람은 이거 안 해" | |
| 1 | "아이디어만 있으면 되지~" 하고 고객 안 만나기 | C섹션 Q2 연결 |
| 2 | 6개월 생활비 없이 퇴사하기 | C섹션 Q6 연결 |
| 3 | 회사 싫어서 도망치듯 창업하기 | C섹션 Q14 연결 |
| CTA | "나는 진짜 준비된 걸까? 체크리스트 해봐" | DecideLab 링크 |

- **예상 길이**: 10~12초
- **포맷**: 빠른 컷 전환 + 강조 텍스트

## 추천 제작 순서
