# 릴스 #2 | 회사 잘 다니는 사람이 절대 안 하는 4가지

## 포맷

- **길이**: 17~20초
- **구조**: 텍스트 카드 빠른 컷 (화면당 3초)
- **글씨**: 굵은 고딕, 화면 중앙, 흰색+검은 테두리
- **BGM**: #1과 같은 사운드 (시리즈 통일)
- **핵심**: #1이 "공감"이면 #2는 "찔림"

---

## 화면 구성

### 화면 1 — HOOK (2초)

```
회사 잘 다니는 사람
이거 절대 안 함
```

> 포인트: "절대"에 빨간색 강조 (시리즈 통일)

## 시리즈 운영

| 순서 | 제목 | 업로드 |
|------|------|--------|
| #1 | 퇴사 고민러편 | 오늘 |
| #2 | 회사 잘 다니는 사람편 | 1~2일 후 |
| #3 | 창업 준비된 사람편 | 3~4일 후 |

- 같은 BGM + 같은 "절대" 빨간 강조 = 시리즈 인식
- #2 캡션에 #1 언급 → 이전 편 조회수 리바운드
- 매 편 CTA는 동일: "프로필 링크에서 무료 진단"
